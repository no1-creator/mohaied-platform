# Sprint 7 — Web Platform (Landing)

هذا الـ Sprint يغطي واجهة منصة الويب لمحايد (صفحة الهبوط التعريفية).

## الهدف

تقديم الموقع للزائر بشكل واضح واحترافي، يشرح فكرة محايد وكيف يحمي حقوق الطرفين، مع دعوات واضحة للتسجيل للعميل ومقدم الخدمة.

## الأقسام الموجودة

1. الهيدر والقائمة العلوية
2. القسم الرئيسي (Hero) مع بطاقة توضيحية
3. كيف تعمل محايد (4 خطوات)
4. المجالات المتاحة الآن
5. للطرفين — العميل ومقدم الخدمة
6. شريط دعوة للتسجيل (CTA)
7. الفوتر

## الملفات

index.html  
styles.css  
README.md

## طريقة الفتح

افتح ملف index.html في متصفح Chrome بعد تحميله على الجهاز.

## المقاسات

- عرض الحاوية: 1200px كحد أقصى.
- التصميم Responsive حتى 560px.
- نفس ألوان الهوية المستخدمة في نسخة الموبايل.

## ملاحظات Product

- الزائر يتصفح كل شيء، وأي أكشن حقيقي يحوله لصفحة التسجيل.
- يجب توضيح دور المشرف كميزة اختيارية.
- يجب إبراز فكرة الحياد وحماية الطرفين في كل قسم.
- المجالات الحالية: برمجة وسوفتوير، هندسة ورسومات، تصميم وواجهات.
- المجالات الجديدة تضاف لاحقًا بنفس نظام الحماية.

## ملاحظات للمطور

- الصفحة ثابتة (Static) في مرحلة النموذج.
- في النسخة الحقيقية ستبنى بـ Next.js + Tailwind.
- روابط الأزرار مؤقتة (#) وتربط لاحقًا بصفحات التسجيل والدخول.
- يجب فصل المحتوى النصي ليسهل ترجمته أو تعديله لاحقًا.

# Sprint 8 — Admin Dashboard

هذا الـ Sprint يغطي لوحة تحكم الأدمن لمنصة محايد.

## الهدف

إعطاء فريق محايد نظرة كاملة على المنصة: المشاريع، المستخدمين، مقدمي الخدمة، المشرفين، الشكاوى، والمدفوعات، مع أدوات مراجعة واتخاذ القرار.

## الأقسام الموجودة

1. القائمة الجانبية (Sidebar)
2. الشريط العلوي (Topbar) مع بحث وإشعارات
3. بطاقات الإحصائيات (KPIs)
4. جدول أحدث المشاريع
5. قائمة الشكاوى الأخيرة
6. طلبات بانتظار المراجعة (توثيق / مشرف / سحب أرباح)

## الملفات

index.html  
styles.css  
README.md

## طريقة الفتح

افتح ملف index.html في متصفح Chrome بعد تحميله على الجهاز.

## المقاسات

- عرض القائمة الجانبية: 260px.
- ارتفاع الشريط العلوي: 64px.
- التصميم Responsive حتى الموبايل مع إخفاء القائمة الجانبية.

## ملاحظات Product

- الأدمن يقدر يشوف كل نشاط المنصة في مكان واحد.
- توثيق مقدمي الخدمة والمشرفين يمر عبر مراجعة يدوية في MVP.
- الشكاوى المصعّدة للتحكيم تظهر بلون مميز.
- طلبات سحب الأرباح تحتاج موافقة قبل التنفيذ.
- كل قرار إداري يجب تسجيله في Activity Log.

## الأقسام الكاملة للوحة (خريطة مستقبلية)

- لوحة التحكم (Overview)
- المشاريع
- المستخدمون
- مقدمو الخدمة
- المشرفون
- الشكاوى والنزاعات
- المدفوعات والسحب
- التقارير
- الإعدادات

## ملاحظات للمطور

- اللوحة ستبنى بـ Next.js + Tailwind في النسخة الحقيقية.
- البيانات في النموذج ثابتة (Mock) وتستبدل لاحقًا بـ API.
- يجب فصل الصلاحيات حسب دور الأدمن (Super Admin / Reviewer / Support).
- كل جدول يحتاج Pagination و Filters في النسخة الحقيقية.
- القرارات الحساسة تحتاج تأكيد وتسجيل كامل.

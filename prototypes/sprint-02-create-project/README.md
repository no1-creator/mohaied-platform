# Sprint 2 — Create Project Flow

هذا الـ Sprint يغطي رحلة إنشاء مشروع داخل منصة محايد من منظور العميل.

## الهدف

تمكين العميل من إنشاء طلب مشروع واضح وموثق، بحيث يستطيع مقدم الخدمة إرسال عرض دقيق، وتستطيع المنصة حفظ مراحل الاتفاق والمتابعة لاحقًا.

## الشاشات الموجودة

1. اختيار المجال
2. اختيار نوع المشروع
3. بيانات المشروع الأساسية
4. وصف المتطلبات
5. الميزانية والمدة
6. المرفقات والروابط
7. طريقة اختيار مقدم الخدمة
8. دعوة شركة أو فريلانسر
9. اختيار المشرف
10. مراجعة الطلب
11. تم إرسال الطلب بنجاح
12. الدعوة بانتظار القبول

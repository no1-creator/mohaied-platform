# Tech Stack Decision

## Frontend Web
Next.js + TypeScript + Tailwind CSS.

## Admin Dashboard
Next.js + TypeScript + Tailwind CSS, with a dashboard-specific component layer.

## Mobile App
React Native / Expo + TypeScript.

## Backend
NestJS + TypeScript.

## Database
PostgreSQL.

## ORM
Prisma.

## Storage
S3-compatible storage such as AWS S3, Cloudflare R2, or Supabase Storage.

## Auth
Custom backend authentication with JWT + Refresh Tokens, later OTP/email/phone verification.
